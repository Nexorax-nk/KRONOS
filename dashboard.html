📚 Ledger Vault

KRONOS-MEMORY-001
Active
Use fixed retry intervals instead of exponential backoff strategy.
Thundering herd effect was identified when dealing with 1000+ concurrent requests, temporarily bringing down the auth service.
📁 governs: atlas/security/auth/authenticator.py
Architect: alice
KRONOS-MEMORY-002
Deprecated
Always use SQLAlchemy async sessions for database queries.
Synchronous queries were blocking the FastAPI event loop, causing severe latency spikes during load. [EVOLVED in https://github.com/Nexorax-nk/KRONOS/pull/4]: We found a better async driver
📁 governs: atlas/database/infrastructure/*.py
Architect: dev-author
KRONOS-MEMORY-003
Active
All database interactions must utilize pooled connection objects. Do not instantiate new connection blocks inside standard queries.
Direct socket instantiation resulted in connection pooling exhaustion under high load during a spike in February, causing 500 server errors for 15 minutes.
📁 governs: atlas/database/infrastructure/connector.py
Architect: charlie
KRONOS-MEMORY-004
Active Security Sentinel
Plaintext logging of PII (credentials, tokens, cards, SSNs) is strictly prohibited. Logger utilities must execute scrubbing routines.
Compliance audits flagged exposed user JWT tokens in the staging server log streams, creating a high-severity data leakage vulnerability.
📁 governs: atlas/core/telemetry/logger.py
Architect: sec_expert
KRONOS-MEMORY-005
Active Security Sentinel
All JWT signature operations must use RS256 asymmetric keys. HS256 symmetric signing is banned due to key-sharing exposure risks.
Prevent credential misuse if a sub-service key leaks. Asymmetric keys ensure only the identity microservice can sign tokens.
📁 governs: atlas/security/auth/authenticator.py
Architect: sec_expert
KRONOS-MEMORY-006
Active Security Sentinel
All database connections must explicitly configure and enforce SSL mode 'require' to secure queries in transit.
Prevent MITM eavesdropping on internal database connection channels inside cloud environments.
📁 governs: atlas/database/infrastructure/connector.py
Architect: charlie
KRONOS-MEMORY-007
Active Security Sentinel
Direct handling of raw credit card numbers or security codes in memory is strictly forbidden. Force payment gateway tokenization.
Avoid PCI-DSS compliance scope inflation and reduce data security breach exposure.
📁 governs: atlas/core/payments/gateway.py
Architect: sec_expert
